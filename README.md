# SiamiRobots

Hi Everyone, this repo will provide you with the information necessary to build the robot pictured below.  

![](Images/car.jpg)

You will need

- For each car
 - Traxxas 4Tec 2.0 chassis (MODEL# 83024-4)
 - Traxxas 5800 may LiPo battery (MODEL# 2843X)
 - Traxxas Brushless Motor - Velineon 3500 (MODEL# 3351R)
 - NVIDIA Jetson TX2 dev kit 
 - Stereolabs ZED2
 - USB hub, 6 port minimum
 - BLDC ESC V6 - Mboards.co
 - Logitech F710 controller
 - 3M double sided tape
 - Portable laptop battery (https://www.amazon.com/gp/product/B015OAJFOC/ref=oh_aui_search_detailpage?ie=UTF8&psc=1)
 - Male 4.0mm to Female 3.5mm ESC adapters - 3x per car
 - Standoff/spacer kit (Amazon - "HELIFOUNER 200 pcs M2.5")
 - USB stick - min 128GB - good quality (Samsung, sandisk, WD, etc...)
 - I will make the platforms necessary to mount all of the hardware

- For the Group
 - Servo connectors https://www.amazon.com/Yiqigou-Extension-Y-Harness-Splitter-Connector/dp/B07PWL648L/ref=sr_1_16?dchild=1&keywords=3+Pin+female+to+female+servo&qid=1611098396&sr=8-16
 - Male TRX to Female XT60 battery adapter multi-pack - 1x per car (Amazon)
 
It will be VERY helpful to have a bluetooth keyboard/mouse as well as an HDMI monitor for programming the robots.
